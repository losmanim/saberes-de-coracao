## Como Vender

### 1. Amazon KDP (Ebooks)

1. Acesse kdp.amazon.com
2. Crie conta (gratuita)
3. Make sure you have to:
   - Capa (PNG, 1600x2560)
   - Arquivo (PDF ou EPUB)
4. Defina preço em reais
5. Publique

**Dicas para Amazon:**
- Use palavras-chave relevantes
- Escolha categorias certas
- Use tráfego externo para promover

### 2. Hotmart (Cursos + Ebooks)

1. Acesse hotmart.com
2. Crie conta vendedor
3. Upload do produto
4. Configure preço
5. Compartilhe link

### 3. Venda Direta (Instagram/WhatsApp)

1. Compartilhe landing page (`sites/landing-vendas.html`)
2. Use link de pagamento (PagSeguro, Mercado Pago)
3. Envie produto por email
