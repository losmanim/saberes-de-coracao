## Navegação por Tag

### Como usar no Obsidian

```markdown
[[#gnose]] - Para linking direto
[[#hermetismo]] 
[[#pratica]]
```

### Buscar por tags

Use a pesquisa do Obsidian com:
- `tag:#espirito`
- `tag:#gnose`
- `tag:#pratica`
- `tag:#ciencia`

## Evolução das Tags

Este sistema de tags é living. À medida que novos conteúdos são adicionados, novas tags podem ser criadas para manter a organização coerente com a filosofia do projeto.
