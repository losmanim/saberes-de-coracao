## Ciclo de Transformação

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│    ┌─────────┐     ┌─────────┐     ┌─────────┐     ┌─────────┐    │
│    │ INCERTO │ →   │ BUSCA   │ →   │ PRÁTICA │ →   │DESPERTAR│    │
│    └─────────┘     └─────────┘     └─────────┘     └─────────┘    │
│        ↑                                                         │
│        └─────────────────────────────────────────────────┘     │
│                         ( ciclo contínuo )                       │
└─────────────────────────────────────────────────────────────────┘
```

### Descrição do Ciclo

1. **INCERTO** — Estado inicial de busca, insatisfação com o mundo material
2. **BUSCA** — Procura por conhecimento, estudo das tradições
3. **PRÁTICA** — Aplicação diário de técnicas (meditação, respiração, hinos)
4. **DESPERTAR** — Expansão de consciência, transformação interior
5. **VOLTA AO INCERTO** — Novo nível de busca, ciclocontinua ascendendo
