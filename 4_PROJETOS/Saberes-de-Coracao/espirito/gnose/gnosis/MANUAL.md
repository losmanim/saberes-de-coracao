## 1. Visão Geral

O **Gnosis** é um site interativo para estudo e prática do conhecimento gnóstico. Combina conteúdo teórico com ferramentas práticas de auto-observação, tudo integrado ao ecossistema **Saberes de Coração**.

### Para que serve
- Estudar os fundamentos da Gnose
- Praticar auto-observação e meditação
- Mapear conexões entre tradições espirituais
- Registar o teu progresso diário de transformação

### Como abrir
Abre o ficheiro `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge). Não requer servidor nem instalação.

## 2. Navegação

O menu fixo no topo dá acesso rápido às principais seções:

```
┌──────────────────────────────────────────────────────────┐
│  ⨁ Gnosis    O que é | Contemplar | Prática | Defeitos │
│              Conexões | Glossário                   [🌙] │
└──────────────────────────────────────────────────────────┘
```

Cada link faz scroll suave até à seção correspondente.

O botão `🌙` / `☀️` no canto direito alterna entre o **modo noturno** (padrão, fundo escuro cósmico) e o **modo diurno** (fundo pergaminho). A preferência fica guardada no navegador.

### 3.10. Práticas

Duas ferramentas práticas:

#### Timer de Auto-observação
```
┌──────────────────────────────────────┐
│            05:00                     │
│                                      │
│  [5 min] [10 min] [15 min] [20 min]  │
│                                      │
│     [ Iniciar ]   [ Reiniciar ]      │
│                                      │
│  ✨ O silêncio é a morada da verdade. │
└──────────────────────────────────────┘
```

- Seleciona a duração (5/10/15/20 min).
- **Iniciar** — começa a contagem decrescente.
- **Pausar** — interrompe (o mesmo botão).
- **Reiniciar** — volta ao tempo inicial.
- Quando o tempo termina, uma frase de sabedoria aparece durante 5 segundos.
- 12 frases diferentes para acompanhar a prática.

#### Respiração Quadrada (4-4-4-4)
```
┌──────────────────────────────────────┐
│          ┌──────────────┐            │
│          │              │            │
│          │   ⬜ animado  │            │
│          │   (svg)      │            │
│          │              │            │
│          └──────────────┘            │
│                                      │
│          Inspira...                  │
│          3 / 5 ciclos                │
│                                      │
│  [5 ciclos] [10 ciclos] [15 ciclos]  │
│                                      │
│            [ Iniciar ]               │
└──────────────────────────────────────┘
```

- O quadrado SVG anima-se a cada fase.
- **Cores por fase:**
  - Inspira → Dourado (#c9a227)
  - Segura → Turquesa (#2a7a8c)
  - Expira → Esmeralda (#2d6a4f)
  - Segura → Ocre (#8b4513)
- Seleciona o número de ciclos (5/10/15).
- **Iniciar/Parar** — controla a prática.

## 4. Funcionalidades Globais

### Modo Dia/Noite
- Botão `🌙` / `☀️` no canto superior direito.
- Alterna entre tema escuro (cosmos) e claro (pergaminho).
- A preferência fica guardada (localStorage).

### Partículas de Fundo
- Partículas douradas animadas representam a "luz do conhecimento".
- Conectam-se entre si quando próximas, como uma rede.
- São apenas visuais e não interferem com a interação.

### Scroll Animado
- As secções revelam-se com animação suave à medida que fazes scroll.
- O título principal tem efeito parallax (desliza mais devagar).

## 5. Dicas de Uso

### Para iniciantes
1. Começa pela secção **"O que é Gnosis"** — clica nos termos destacados para aprender o vocabulário.
2. Vai ao **Glossário** e explora todos os termos.
3. Lê os **Ensinamentos Centrais** — clica em cada card.
4. Pratica 5 minutos com o **Timer de Auto-observação**.

### Para praticantes diários
1. Abre o site e vai à **Contemplação** — deixa a frase do dia ressoar.
2. Ao final do dia, usa a **Roda dos 3 Fatores** para registar as tuas observações.
3. Consulta os **6 Defeitos** quando notares um padrão emocional.
4. Usa a **Respiração Quadrada** sempre que precisares de centrar.

### Para estudo aprofundado
1. Explora a **Cadeia de Causa e Efeito** — reflete em cada elo.
2. Estuda as **Correspondências** — vê o macro no micro.
3. Navega as **Conexões entre Tradições** — percebe a unidade por trás da diversidade.

### Dados guardados no navegador
O site guarda localmente (sem servidor):
- **Tema** (modo dia/noite)
- **Respostas da Roda dos 3 Fatores** (organizadas por data)
- **Contagem de observações dos Defeitos**

Para limpar estes dados: Configurações do navegador → Privacidade → Limpar dados do site.
