# 🪞 Lei da Correspondência - "Como em Cima, Assim em Baixo"

> O terceiro princípio hermético revela a unidade fundamental de toda a existência.

---

## O Princípio

> *"Como é em cima, assim é embaixo; como é embaixo, assim é em cima."*
> — Tábua de Esmeralda, Hermes Trismegisto

### Definição simples

Tudo o que existe no macrocosmo existe no microcosmo. O universo é um espelho infinito onde cada parte reflete o todo.

---

## Aplicações Práticas

### No Corpo Humano

```
COSMO              CORPO HUMANO
─────────────────────────────────────────────
Sistema Solar    →   Sistema Circulatório
Planetas         →   Órgãos
Lua             →   Lua Cerebral (glândula pineal)
Estrelas        →   Células
Galáxias        →   Cérebro
```

### Nas Emoções

```
Emoção Superior   →   Emoção Inferior
─────────────────────────────────────────────
Amor divino      →   Amor humano
Sabedoria        →   Conhecimento
Paz eternal     →   Paz momentânea
Força criadora   →   Força muscular
```

### Nos Ciclos

```
Ciclo Cósmico    →   Ciclo Humano
─────────────────────────────────────────────
Ano solar        →   Ciclo circadiano
Lua crescente   →   Crescimento
Lua minguante   →   Decadência
Estações        →   Ciclos de vida
```

## Como Usar Este Princípio

### 1. Para Healing (Cura)

Se há um padrão em qualquer nível, você pode influenciá-lo em qualquer nível:
- Cura física? Trabalhe o emocional
- Cura emocional? Trabalhe o mental
- Cura mental? Trabalhe o espiritual

### 2. Para Manifestação

Visualize o que deseja no nível mais elevado, e o nível inferior se ajustará:
- Deseja saúde? Visualize perfeição emocional
- Deseja abundância? Visualize fluidez cósmica
- Deseja amor? Visualize amor universal

### 3. Para Compreensão

O que está fora está dentro:
- Problema no mundo? Examine seu interior
- Conflito externo? Conflito interno
- O que você condena em outros, você também tem

## Prática: Meditação da Correspondência

```
1. Sente-se em silêncio
2. escolha um aspecto da natureza
3. pergunta: "O que isto reflete em mim?"
4. Observe a resposta que surge
5. Pergunte: "O que isto reflete no cosmos?"
6. Integre a compreensão
```
