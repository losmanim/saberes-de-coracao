# 🖤 Saberes de Coração

> "O verdadeiro conhecimento deve ser passado de coração, com boa vontade, em empatia ativa, elevando o campo."

## 🌟 O Que Este Projeto Representa

Uma estrutura organizada de conhecimentos espiritual, filosófico e científico, compilados com intenção de elevar quem os recebe.

**Este não é um projeto de vendas.** É um projeto de compartilhamento genuíno.

## 💚 Filosofia

> "Quando compartilhamos conhecimento com intenção pura, elevamos não apenas quem recebe, mas também quem oferece. O campo coletivo se expande."

### Princípios do Compartilhamento

1. **Sem expectativa de retorno** — O conhecimento é livre
2. **Com empatia** — Cada pessoa tem seu ritmo
3. **Elevando o campo** — Contribuir para a evolução coletiva
4. **De coração** — Não por obrigação, mas por amor

## 🌐 Site Modelo

Um exemplo completo de como os saberes podem ser apresentados interativamente, com busca, filtros, modal de conteúdo e player de multimídia.

---

## 🧭 Como usar este repositório

- `Saberes_de_Coracao-site-3.0/` é o site canônico. Mantenha essa versão como base de edição ativa.
- `database/dados-unificados.json` é a fonte de verdade para o site 3.0. Atualize esse arquivo para incluir novos saberes e multimídia.
- Conteúdos temáticos também podem ser adicionados nas pastas `espirito/`, `practica/`, `ciencia/`, `viver/` e `vida-verdadeira/`.
- Documentos de apoio:
  - `CONTRIBUTING.md`
  - `ARCHITECTURE.md`
  - `ROADMAP.md`
  - `_indice.md`
  - `_grafo.md`
  - `_tags.md`
  - `_guia-didatico.md`
