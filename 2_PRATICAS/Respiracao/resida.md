## Por Que Respirar?

A respiração é a função mais primal e a mais divina. É o movimento contínuo entre o céu e a terra, entre o espírito e a matéria.

### Benefícios

| Nível          | Benefício                                |
| -------------- | ---------------------------------------- |
| **Físico**     | Oxigenação celular, relaxamento muscular |
| **Energético** | Circulação do prana/kundalini            |
| **Mental**     | Calma, clareza, foco                     |
| **Espiritual** | Conexão com o sopro divino               |

## Técnicas Fundamentais

### 1. Respiração Quadrada (Samavritti)

```
INHALAR   → Segurar  → EXALAR  → Segurar
  4s        4s        4s       4s
```

**Quando usar:** Para acalmar a mente, meditação, ansiedade

### 2. Respiração Alternada (Nadi Shodhana)

```
Nariz esquerdo cerrado → Inalar pelo direito
Nariz direito fechado → Segurar
Exalar pelo esquerdo
Inalar pelo esquerdo → Segurar → Exalar pelo direito
```

**Quando usar:** Equilibrar hemispérios, elevar kundalini

### 3. Respiração de Fogo (Kapalabhati)

```
Exalar forte pelo nariz (passivo)
Inalar passivamente
Repetir em ritmo rápido
```

**Quando usar:** Energizar corpo, limpar canales

### 4. Respiração do Sopro (Sitali)

```
Enrolar a língua em tubo
Inalar pelo tubo
Segurar
Exalar pelo nariz
```

**Quando usar:** Resfriamento, paz interior

## Ciclo de Respirações

### Manhã (Despertar)

```
1. Kapalabhati - 1 minuto (energizar)
2. Respiração profunda - 3 minutos (estabelecer)
```

### Meio-dia (Centro)

```
1. Nadi Shodhana - 5 minutos (equilibrar)
2. Respiração quadrada - 3 minutos (centrar)
```

### Noite (Integração)

```
1. Respiração profunda - 5 minutos
2. Sitali - 3 minutos (calmar)
```
